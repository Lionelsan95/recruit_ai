# Ice Breaker

To launch the application, you should: 
1. create a .env file
2. update the content of the .env file by adding 3 variables : 
    - OPENAI_API_KEY
    - PROXYCURL_LINKEDIN_API_URL
    - PROXYCURL_API_KEY
    - TAVILY_API_KEY

proxycurl : https://nubela.co/proxycurl/
tavily api : https://tavily.com/
